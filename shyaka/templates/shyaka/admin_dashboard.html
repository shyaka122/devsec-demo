Authorization Model

Role Hierarchy

  • 🔑 Admin: Full system access, user management, role assignment
  • 👔 Staff: Elevated permissions, moderation capabilities
  • 👤 User: Standard authenticated user permissions
  • 👁 Anonymous: Limited access, registration and login only

Access Control Strategy

All privileged views enforce role-based authorization using Django decorators and group membership. Unauthorized access attempts return HTTP 403 Forbidden with appropriate error messages.
